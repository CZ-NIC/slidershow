Tutorial

Let's take a look on some features.
Hit PageDown to proceed.

Every <article> tag corresponds to a frame.

Hit Esc any time to toggle the menu.

Put there a photo.
If the photo is too wide, a panorama plays automatically.
Or you can set manually the points to zoom in on a photo.
The photo geo information starts the HUD map in the corner.
Map can be on fullscreen, too.
Convert a GPX route to the frames.
Video playing made easy.
Have you realized we trimmed?
Cut is as easy as adding a number into the video URL hash.
Or change the speed.

You have seen some cool features.
Check the presentation source code with Ctrl+U.
Cool and clean, isn't it?

Remember the Esc menu? There are many things you can do...

  • See the thumbnail ribbon (j)
  • See the thumbnail grid (g)
  • Launch an auxiliary window to be displayed on a second monitor during the presentation
  • Start tagging mode to help you organize the photos for a better presentations

Bookmark the SlideRshow to build a smooth presentation.
